Локальна версія сайту для вивчення мови програмування Python.


✅ Створюємо порожню теку python_learner_local.

Зберігаємо в неї всі файли:

Відкриваємо index.html в браузері (Chrome/Firefox). Pyodide підвантажиться з CDN — може зайняти ~10-30s при першому завантаженні.

Примітка: Pyodide працює повністю в браузері, tester.py підвантажується через fetch('tester.py'),
тобто файл має бути доступний у тій же теці, де лежить index.html.
Якщо браузер блокує file:// доступи, запустити простий локальний сервер (наприклад в консолі (cmd), 
    python -m http.server (в теці з файлами) 
    і відкрити http://localhost:8080
Або:
    cmd.exe -> cd C:\Users\User-Name\Desktop\python_learner_local && python -m http.server 8080
    і відкрити http://localhost:8080

Корисні команди для сервера при проблемах з портмами
    # 1. Знайти процес, що займає порт 8080
    netstat -ano | findstr :8080

    # 2. Запам'ятайте PID (ідентифікатор процесу)
    # 3. Примусово зупинити процес
    taskkill /PID <PID> /F

Для встановлення інтерпретатора Python на ПК (самої мови Python) потрібно завантажити інсталятор з офіційного сайту та запустити його.

Ось покрокова інструкція для найпоширеніших операційних систем:
    1. Завантаження інсталятора
        Перейдіть на офіційний сайт Python: https://www.python.org/downloads/
        Сайт автоматично визначить вашу операційну систему (Windows, macOS, Linux).
        Натисніть кнопку "Download Python [Остання версія]" (наприклад, "Download Python 3.12.0").
    2. Встановлення на Windows
        Встановлення на Windows вимагає одного дуже важливого кроку, який гарантує,
        що ви зможете запускати Python з терміналу (Командного рядка/CMD):
            Запустіть завантажений файл-інсталятор (.exe).
            У першому вікні інсталятора ОБОВ'ЯЗКОВО поставте галочку біля пункту:
            ✅ Add python.exe to PATH
            Це критично важливо, оскільки дозволить вам викликати команди python та pip з будь-якої директорії у терміналі.
            Натисніть "Install Now" (Встановити зараз) і дочекайтеся завершення процесу.

Перевірка встановленої версії.
    Перевірка: Відкрийте термінал (або командний рядок) і виконайте команди:
        cmd -> python --version
    # Або
        cmd -> python3 --version
Перевірка встановлення
Після завершення встановлення відкрийте Термінал (CMD/PowerShell у Windows або Terminal у macOS/Linux) і виконайте наступну команду:
    python --version
    Або (якщо виникає помилка):
    python3 --version

Якщо встановлення пройшло успішно, ви побачите номер встановленої версії (наприклад, Python 3.12.0).
Це означає, що інтерпретатор Python готовий до роботи.

Встановлення бібліотек Python:
    pip install flask-cors (це назва бібліотеки)
